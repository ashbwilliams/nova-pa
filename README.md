# NOVA Performing Arts

Public website for NOVA Performing Arts, a Central Texas 501(c)(3) nonprofit expanding access to youth performing arts education.

The site introduces NOVA's flagship noncompetitive marching percussion academy, explains the access challenge it addresses, and provides clear paths for students, families, educators, donors, and community partners.

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
